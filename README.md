# Battle Cats Randomizer (BCR)

Battle Cats Randomizer is a tool that generates a randomized mod of The Battle Cats, altering enemy and unit traits, abilities, and other gameplay elements to create chaotic and unique playthroughs. More randomization options to come in the future!

## Features

Enemy trait randomization

Unit ability / talents randomization

Highly customizable config file

And more to come!

## Requirements

Python 3.11+

An APK file for The Battle Cats
